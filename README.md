# Signal-Modulation-Recognition-GUI
## 项目简介
基于Python+PyQt5开发的通信信号智能侦察可视化系统。
模拟无线信道环境，生成ASK/PSK/FSK/16QAM四种调制信号，叠加不同信噪比噪声；提取时域、频域多维信号特征，使用SVM机器学习模型完成自动调制方式识别。配套可视化界面展示基带波形、射频时域波形、功率谱，并支持模型离线训练、性能评估、实时信号识别，适合通信信息化实施、无线电信号侦察原型演示。

## 技术栈
Python、PyQt5、NumPy、Matplotlib、SciPy、Scikit-learn(SVM)

## 项目文件说明
- main.py ：程序入口，包含信号生成、特征提取、GUI交互、模型训练预测全部代码
- requirements.txt：项目Python依赖清单
- README.md：项目说明文档

## 实现功能
1. 仿真生成ASK、PSK、FSK、16QAM数字调制射频信号
2. 可自由配置信噪比SNR，模拟真实信道噪声环境
3. 提取幅度方差、峰均比、峭度、信息熵、频谱特征等7维信号指纹
4. 可视化展示数字基带波形、射频时域波形、信号功率谱密度
5. 离线批量生成数据集，训练SVM分类模型
6. 多信噪比条件下模型精度测试，输出识别准确率曲线与混淆矩阵
7. 单样本信号实时调制类型智能识别

## 项目收获
掌握数字通信调制原理、无线信道仿真、信号特征工程、机器学习分类流程；熟练使用PyQt5开发可视化工程软件，具备仿真系统部署、调试、工程交付相关能力。
